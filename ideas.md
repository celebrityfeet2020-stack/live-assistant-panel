# 直播讲解助手控制面板 - 设计构思 (修订版)

## 1. 设计风格选择：深色护眼科技风 (Dark Eye-Care Tech)

**核心理念**: 打造一个专业、沉稳且适合长时间注视的直播控制台。界面采用深色调，避免高对比度和刺眼的色彩，注重信息的清晰呈现和操作的舒适性。

**色彩哲学**:
- **背景**: 深灰色系 (Deep Zinc/Slate)，如 `#18181b` (Zinc 950) 或 `#0f172a` (Slate 950)，提供稳重的视觉基础。
- **卡片背景**: 稍浅的深灰色，如 `#27272a` (Zinc 800)，与背景形成微妙层次。
- **强调色**: 柔和的靛蓝 (Indigo) 或 天蓝 (Sky Blue)，如 `#6366f1` 或 `#0ea5e9`，用于激活状态，避免使用高饱和度的霓虹色。
- **文本**: 灰白色 (Off-White)，如 `#e4e4e7` (Zinc 200) 用于主要文字，`#a1a1aa` (Zinc 400) 用于次要文字，降低对比度刺激。

**布局范式**:
- **卡片式网格布局**: 保持卡片式设计，每个链接配置作为一个独立的模块。
- **柔和边界**: 使用圆角和微弱的边框 (`border-white/10`) 来界定区域，而非强烈的阴影或发光。
- **居中与留白**: 保持适当的页面边距，避免信息过密。

**交互哲学**:
- **舒适反馈**: 交互动画保持平滑、舒缓，避免突兀的闪烁。
- **清晰指引**: 按钮和输入框的状态变化清晰但柔和。

**排版系统**:
- **字体**: 使用 `Inter` 或系统默认无衬线字体，保证最佳的可读性。
- **字重**: 避免使用极细或极粗的字重，保持中等字重以减轻阅读负担。

## 2. 关键组件设计

1.  **链接配置卡片**:
    - **外观**: 深色背景，圆角矩形，微弱边框。
    - **标题**: "链接 01", "链接 02"... 使用柔和的亮色。
    - **关键词标签**: 胶囊状标签，背景色为半透明的强调色，文字清晰。
    - **输入框**: 深色背景，无边框或微弱边框，聚焦时柔和高亮。

2.  **添加链接按钮**:
    - 位于列表末尾，虚线边框，悬停时背景轻微变亮。

3.  **顶部导航栏**:
    - 包含标题 "直播讲解助手" 和全局操作按钮（保存）。
    - 使用玻璃拟态 (Backdrop Blur) 效果，但保持深色底色。

## 3. 视觉资源

- **背景图**: 生成一张抽象的、深色系的、带有流线型或几何元素的背景图，色彩偏向深蓝或深紫，亮度极低，作为页面的底层纹理。

## 4. 选定方案

我们将采用 **"深色护眼科技风"**。

**具体执行**:
- Tailwind 颜色配置：主背景 `bg-zinc-950`，组件背景 `bg-zinc-900`，文字 `text-zinc-200`。
- 强调色：`indigo-500`。
- 字体：`Inter`。
